## 🥕 코드 업로드 방법

1) 팀 Origin Repository Clone 진행
2) 로컬 Repository에서 본인의 이름과 동일한 브랜치를 생성 후 Checkout (main -> 본인 이름)
3) 본인 이름에 해당하는 패키지에서 문제 풀이 진행
4) push 완료 후 팀 Origin Repository에서 Pull Request(PR) 버튼 클릭
5) merge 후 branch 삭제
